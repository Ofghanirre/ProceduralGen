## Compilation & Launch
To compile this project you will have to follow these steps:
- Install a cpp compiler like **g++**
- Install **cmake**
- Go Into the `build` folder
> $ cd build
- Generate the make file
> $ cmake ..
- Build the Project
> $ cmake --build .
- Start the project
> $ proceduralGen
