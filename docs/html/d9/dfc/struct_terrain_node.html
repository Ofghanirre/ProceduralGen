Detailed Description

A Terrain is a collection of NoiseGenerator, each of them being mapped to a function The function is used to punderate the noise generated by the NoiseGenerator.

The predicate is a function applyable to the old value present in before the execution of this TerrainNode It can be usefull if you want to apply the terrainNode NoiseGenerator based on some previous value
